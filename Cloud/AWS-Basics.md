# AWS Basics

## Overview

AWS (Amazon Web Services) is one of the largest cloud computing platforms in the world.

Organizations use AWS to:
- host applications;
- store data;
- manage infrastructure;
- improve scalability and availability.

---

# Core AWS Services

## EC2

EC2 (Elastic Compute Cloud) provides virtual servers in the cloud.

Organizations use EC2 to:
- host applications;
- run Linux or Windows systems;
- manage cloud infrastructure.

---

## S3

S3 (Simple Storage Service) is cloud storage.

It is used for:
- backups;
- file storage;
- logs;
- application data.

### Risk

One of the most common cloud risks is:
- public S3 buckets.

This may lead to:
- data leakage;
- unauthorized access;
- compliance violations.

---

## IAM

IAM (Identity and Access Management) controls:
- users;
- permissions;
- access rights.

IAM helps organizations implement:
- least privilege;
- access control;
- identity management.

---

## CloudWatch

CloudWatch is used for:
- monitoring;
- logging;
- alerts;
- operational visibility.

---

# AWS Security Perspective

Cloud security depends heavily on:
- proper configuration;
- strong IAM policies;
- MFA;
- monitoring;
- logging.

---

# Main Lesson

AWS provides powerful infrastructure, but security responsibilities are shared between the provider and the customer.
