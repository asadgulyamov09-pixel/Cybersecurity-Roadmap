# Azure Basics & Cloud Governance

## Overview

Microsoft Azure is a cloud computing platform used by organizations to manage infrastructure, applications, storage, networking, and identity systems.

Azure is widely used in enterprise environments because of its integration with Microsoft technologies and corporate systems.

---

# Azure Services

Azure provides:
- virtual machines;
- storage;
- networking;
- identity management;
- cloud security tools.

Organizations use Azure to:
- host applications;
- manage cloud infrastructure;
- improve scalability and availability.

---

# Cloud Governance

Cloud governance means creating rules, controls, and policies for managing cloud environments securely and efficiently.

Governance helps organizations:
- control access;
- reduce risks;
- maintain compliance;
- manage cloud resources.

---

# Identity & Access Management

Identity management controls:
- users;
- permissions;
- authentication;
- access rights.

Strong identity management helps reduce:
- unauthorized access;
- insider threats;
- privilege abuse.

---

# Common Cloud Risks

| Risk | Business Impact |
|---|---|
| weak permissions | unauthorized access |
| no MFA | account compromise |
| misconfiguration | data leakage |
| no logging | undetected attacks |

---

# Security Controls

Important cloud controls include:
- MFA;
- least privilege;
- monitoring;
- logging;
- access reviews.

---

# Main Lesson

Cloud security depends heavily on governance, identity management, and proper access control.
